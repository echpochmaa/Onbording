Где мы обитаем и общаемся

Основной мессенджер — Mattermost. Telegram используется только для информационных каналов.

📢 Telegram-каналы (только информационные)
  • MC D Info — основной канал с новостями и анонсами
  • MC D: Аналитика — канал для аналитических новостей и отчетов
⚠️ Важные правила регистрации в Mattermost
  • Username (ник) — строго по корпоративной почте, часть до @. Пример: al.ivanov@rambler-co.ru → al.ivanov
  • Full name — обязательно заполните. Имя и Фамилия на кириллице. Пример: Иван Петров
  • Аватарка — обязательна для всех (актуальное фото, не старше 2 лет)
📝 Регистрация по пригласительной ссылке
  1. Открой пригласительную ссылку
  2. Заполни форму: Email (корпоративный), Username (часть email до @), Password
  3. На корпоративный email придёт письмо для подтверждения
  4. Перейди по ссылке в письме, чтобы активировать аккаунт

После регистрации — напиши Веронике или руководителю, чтобы тебя добавили в рабочие каналы.
