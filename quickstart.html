Вопросы получения доступов для работы DevOps

1 Получить VPN

Для начала тебе нужно получить OpenVPN конфиги на Dev/Stage окружения.

  • подключение к подам на окружениях через lens/kubectl
  • работа с БД
  • доступ на стенды веба или Kafka
  • dev vpn используется также для инфраструктуры (доступ в gitlab, vault)

Подробнее про dev контур: Инфраструктура Advanced

2 Доступы к Git

После того, как появится доступ в gitlab, через своего Лида или через команду DevOps получаем доступы

DevOps репозитории
products (общий) sd-helm sd-ansible sd-terraform

3 Jira, Confluence, DevOps портал

Данные инструменты расположены в Sberworks, для них нужна учётка SberteamID. Инструменты доступны через dev vpn (ipsec тоннель из Advanced облака в Sberworks). С доступами поможет команда.
